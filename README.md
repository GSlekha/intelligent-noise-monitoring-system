# Intelligent Noise Monitoring and Horn Detection System

## Overview
This project is an embedded edge-based intelligent noise monitoring prototype developed using Arduino UNO and audio signal preprocessing concepts for real-time horn event detection.

The system explores smart traffic monitoring and urban IoT applications using lightweight embedded processing techniques.
